# Salt Worker Weather App 🌤️🧂

url:https://uppala-vanna-vedic.base44.app
A smart weather monitoring and safety alert system specially designed for salt pan workers.

## 📌 Features
- Live weather updates
- Heat & rain alerts
- Push notifications
- Tamil voice/text safety messages
- Salt area location support
- Worker protection guidelines

## 🎯 Purpose
Salt workers work in extreme heat and harsh weather conditions. 
This app helps them stay safe by providing early weather warnings and protective instructions.

## 🛠️ Tech Stack
- HTML / CSS / JavaScript
- Weather API
- Mobile responsive design

## 👩‍💻 Developed For
Improving health, safety, and productivity of salt workers.
